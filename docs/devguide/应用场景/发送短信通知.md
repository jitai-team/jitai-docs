# 发送短信通知

某制造企业需要建立设备故障自动告警机制，当生产设备出现异常或故障时，系统立即发送短信通知运维团队和车间负责人。传统的现场巡检发现故障滞后，设备监控系统的邮件告警容易被忽略，导致故障处理不及时影响生产进度。

企业要求故障告警短信能够包含设备编号、故障类型和发生时间等关键信息，支持同时通知多名相关人员，确保设备故障能够被及时发现和快速响应，最大程度减少生产损失。
