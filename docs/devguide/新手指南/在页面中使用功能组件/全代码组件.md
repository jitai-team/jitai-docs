# 全代码组件

## 全代码组件

全代码组件是一种特殊类型的组件，它可以实现任何功能，但需要编写完整的组件逻辑，通过自定义界面、逻辑、样式，以及对模型数据的各种操作，来扩展功能。全代码组件基于React开发，因此需要熟悉React开发知识。

### 创建全代码组件
![全代码组件创建](./img/15/全代码组件创建.gif)
在可视化页面编辑器中选择全代码组件，将全代码组件拖拽到页面画布中，就生成了一个全代码组件。
