# 角色权限

## 创建应用角色

**侧重特性：**

*   创建不同角色并提供不同的门户导航菜单

## 配置数据权限

**侧重特性：**

*   给角色配置数据表模型的增删改查及数据范围权限

## 配置功能权限

**侧重特性：**

*   给角色配置页面中组件的按钮权限

*   给角色配置页面中组件的数据范围及字段操作权限

## 角色层级管理

**侧重特性：**

*   配置角色可以管理的其它角色

## 角色分配管理

**侧重特性：**

*   在管理门户中为组织架构成员分配应用角色
