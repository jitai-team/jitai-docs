# 让 Agent 操作数据库

在 JitAI 应用中，数据模型对应数据库中的表或视图。Agent 可以通过数据模型对数据库执行增删改查操作，也可以调用模型上定义的自定义函数。

## 配置数据模型

在 Agent 编辑器的 `Models` 页签中，支持三种模式：

- **智能模式**：Agent 自动感知应用中的所有数据模型，自行选择并调用。适用于开放性场景，特定目标场景下通常不使用。
- **自定义模式**：按需勾选目标数据模型，精确控制 Agent 可操作的数据范围。
- **禁用**：Agent 不会感知任何数据模型的存在，也不会调用数据模型。

## 内置函数与自定义函数

- **内置函数**：每个数据模型自带的通用增删改查（CRUD）函数，开箱即用
- **自定义函数**：按业务需求封装的逻辑，如复杂查询、数据校验、状态变更、级联处理等

## 典型场景

- "帮我查最近一周的订单"
- "把这条客户信息更新为已签约"
- "统计上个月的销售额"
