# 按需启用工具函数

每个数据模型、服务函数、页面元素都包含多个函数。开发者可以根据实际需求选择性启用需要的函数，而不是全部对外开放。

## 启用和关闭函数

在 Agent 编辑器的`Models`、`Functions`或`工作页面`中，每个函数右侧都有启用开关：

- 按需启用需要的函数
- 关闭不需要的函数

:::warning
不要简单粗暴地启用全部函数，要根据实际需要针对性启用函数，否则会增加不必要的 Token 消耗、给大模型带来理解负担、引入不必要的操作风险。
:::
