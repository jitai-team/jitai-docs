# 让 Agent 操作页面组件

为 Agent 指定工作页面后，可以将页面上的函数配置为 Agent 可调用的工具，让 AI 操作前端界面组件。

<!-- TODO: 配图 - GIF：选择工作页面并启用页面函数 -->

## 配置工作页面

在 Agent 编辑器 `更多` → `工作页面` 中：

1. 选择目标页面
2. 页面函数列表自动加载
3. 按需勾选需要启用的函数

可用的函数包括：

- **内置函数**：获取页面变量、刷新表格、打开弹窗等
- **组件函数**：表格排序、表单提交等
- **自定义页面函数**：开发者自行定义的函数

## 在用户门户中验证

页面通过门户菜单供用户访问。Agent 对页面函数的调用需要在页面的运行时上下文中执行，因此在开发者门户中无法验证，需切换到对应的用户门户中测试。

## 典型场景

- "帮我刷新这个表格"
- "打开新建客户的弹窗"
- "切换到上个月的报表页面"
