# 让 Agent 调用服务函数

服务函数用于封装复杂的业务逻辑处理过程。添加为 Agent 工具后，AI 就能直接调用这些函数，**是优化 Agent 执行能力和准确度的重要手段**。

<!-- TODO: 配图 - GIF：添加服务函数工具 -->

## 配置服务函数

在 Agent 编辑器的 `Functions` 页签中，支持三种模式：

- **智能模式**：Agent 自动感知应用中的所有服务函数，自行选择并调用。适用于开放性场景，特定目标场景下通常不使用。
- **自定义模式**：按需勾选目标服务函数，精确控制 Agent 可调用的函数范围。
- **禁用**：Agent 不会感知任何服务函数的存在，也不会调用服务函数。

## 让大模型准确理解你的函数

清晰无歧义的函数名称、函数描述、参数声明，可以提高大模型的理解准确度，更精准地完成任务。

:::tip
函数名称、描述和参数声明是大模型理解工具能力的关键依据，务必准确清晰。
:::

## 典型场景

- 订单处理：创建订单、取消订单、查询物流
- 数据计算：统计分析、报表生成
- 消息推送：发送通知、邮件提醒
