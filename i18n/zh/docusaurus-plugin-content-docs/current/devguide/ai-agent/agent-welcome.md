# 配置欢迎语和预置问题

在 Agent 编辑器中切换到`更多` → `欢迎语 & 预置问题`，可配置用户打开聊天页面时的首次体验。良好的欢迎语和预置问题能让用户快速理解 Agent 的能力范围，无需思考如何提问。

## 欢迎语

用户打开 Agent 聊天页面时看到的第一句话。默认为"有什么我能帮你的吗？"，可根据 Agent 的定位自定义。

欢迎语应简洁说明 Agent 的身份和能力，让用户一眼就知道这个 Agent 能做什么。

## 预置问题

预置问题展示在欢迎语下方，用户点击即可快速发起对话，无需手动输入。支持添加多个预置问题，每个问题都可随时修改或删除。

预置问题应覆盖用户最可能提出的典型需求，引导用户快速上手。

## 典型配置示例

- **订单助手**
  - 欢迎语：`你好，我是订单助手，可以帮你查询、修改和取消订单。`
  - 预置问题：`查询我的最新订单`、`取消最近一笔订单`、`修改收货地址`

- **数据分析 Agent**
  - 欢迎语：`你好，我是数据分析助手，可以帮你分析销售数据和生成报表。`
  - 预置问题：`本月销售趋势如何？`、`按地区分析客户分布`、`对比上月和本月销售额`
