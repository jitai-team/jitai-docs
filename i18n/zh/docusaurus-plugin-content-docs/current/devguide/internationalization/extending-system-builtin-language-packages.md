# 扩展平台内置语言包

平台内置 **简体中文(zhCN)** 和 **English(EN)** 两种语言包。

开发者可以在其应用内创建同名的语言包来扩展或覆盖内置语言包。
![overwrite-pack](./img/overwrite-pack.png)

在自定义语言包内，您可以替换原有翻译内容或添加新的词条，实现更精细的本地化控制。

![overwrite-terms](./img/overwrite-terms.gif)
